              Super Admin (Company Owner / System Controller) 
    Can Access 
Create / Manage HR accounts 
Create / Manage Managers 
View all employees

Configure leave types & policies 
View all reports (Payroll, Attendance, Leave, Performance) 
View audit logs 
Activate / deactivate users 
Multi-company management (if you implement) 
   Cannot (Normally Should Not) 
Approve daily leave requests 
Process monthly payroll manually 
Mark attendance 
Apply leave as employee 
    Super Admin = System Control + Monitoring 
Not daily operations.
